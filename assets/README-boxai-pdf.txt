Box AI pitch PDF
------------------

The site links to:  assets/boxai-pitch.pdf

1. Copy your PDF from Documents (or anywhere) into this folder and name it exactly:
   boxai-pitch.pdf

   Or run from the project root in PowerShell:
   .\scripts\copy-boxai-pitch-to-assets.ps1 -Source "FULL_PATH_TO_YOUR_FILE.pdf"

2. Commit git add assets/boxai-pitch.pdf and push so Vercel/your host serves it.

The download button on the Box AI page uses href="assets/boxai-pitch.pdf".
