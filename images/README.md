# Site images

Put image files in **images/projects/** so **Search Gapsitters** and **Search Projects** show real photos. (The old site used this single folder for both.)

## Search Gapsitters (sitter photos)

- `images/projects/elizabeth.jpg`
- `images/projects/danielle.jpg`
- `images/projects/john.PNG`
- `images/projects/violet.jpg`
- `images/projects/nichole.jpg`
- `images/projects/anna.jpg`

## Search Projects (property/house photos)

- `images/projects/johnhouse.jpeg`
- `images/projects/dorothyhouse.jpg`
- `images/projects/stairsrepair.jpg`
- `images/projects/stairsfixed.JPG`
- `images/projects/dogswelcomedownedtree.jpg`

Use the exact filenames above. JPG, PNG, or WebP are fine. If a file is missing, a gray placeholder with the name will show instead.
