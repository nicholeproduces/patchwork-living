HEIC → JPG (for IMG_1396 / downed tree backyard)
------------------------------------------------
Your file IMG_1396.heic must be converted to JPG for browsers to show it.

On Windows (PC):
  1. Install "HEIF Image Extensions" from Microsoft Store (free).
  2. Open IMG_1396.heic in the Photos app (or Paint).
  3. Save As → JPEG, and save as: img_1396.jpg in this folder.
  Or use a free site: heictojpg.com or convertio.co (upload HEIC, download JPG).

On Mac:
  Open the HEIC in Preview → File → Export → Format: JPEG → Save as img_1396.jpg.
  Put img_1396.jpg in this folder (images/projects/).
